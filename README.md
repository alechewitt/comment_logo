Helper utility to generate fancy logos for the comments in code.

<h1>Example Output</h1>
```
=       ======   =====    ======           ======   ======   ===  ===   ===  ===   =====   ==   =   =====   
=       =    =   =        =    =           =        =    =   = =  = =   = =  = =   =       = =  =     =     
=       =    =   =  ===   =    =           =        =    =   =  ==  =   =  ==  =   =====   =  = =     =     
=       =    =   =   =    =    =           =        =    =   =      =   =      =   =       =   ==     =     
=====   ======   =====    ======           ======   ======   =      =   =      =   =====   =    =     =     
```

<h1>Usage</h1>
```
npm install
node logoGenerator.js -w "logo comment"
```
